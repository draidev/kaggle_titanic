## 2. 데이터 분석
1. SibSp 숫자에 따라 생존률이 차등이 있었다
2. Parch 숫자에 따라 생존률이 차등이 있었다
3. Pclass는 낮은 클라스 일수록 사망률이 높았다. (3rd > 2nd > 1st)
4.성별에 따라 생존률이 차등이 있었다.
5. 탑승장소에 따라 생존률 차등이 있었다.
6. 운임은 폭이 너무 커 log로 변환한 뒤 확인이 필요하다
